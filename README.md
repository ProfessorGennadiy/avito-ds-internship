# ***Тестовое задание 3: Прогнозирование откликов***

# Контекст задачи
На платформе Авито Работа происходит мэтчинг двух сторон: соискателя, который ищет работу, и работодателя, который размещает вакансию.

Когда соискатель вводит поисковый запрос, система показывает ему список подходящих вакансий. Нам важно понимать, какова вероятность того, что конкретный пользователь откликнется на конкретную вакансию.
Контакт (item_contact) — это ключевое событие, звонок или сообщение работодателю через платформу.

# Задача
Предсказать совершит ли соискатель контакт (1) или нет (0).

# Особенности датасета
Сильный дисбаланс классов(6.28% совершивших контакт)
Разные типы данных - временные, числовые, текстовые, категориальные

# Решение
# EDA
Выявлены дубликаты (22%)
Пропуски только в query_mcat(принято решение заполнить цифрой 0)

# Feature Engineering
Созданы признаки с временными паттернами - is_night/is_morning/is_evening, с днем недели и месяца - day_of_week, day_of_month, и с паттерном сезонности - is_spring, is_winter, is_summer
Созданы признаки с наибольшим влиянием - совпадение локации, категории, популярность вакансии и работодателя

# Выбор модели
CatBoost был выбран по следующим причинам - работа с категориальными данными, встроенная обработка дисбаланса, устойчивость к переобучению

Результат - AUC = 0.7623
